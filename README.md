
# Bank Deposit Time Series Forecasting

## Project Overview

This project developed a univariate time series forecasting model for total deposits held at U.S. commercial banks using monthly data from the Federal Reserve Economic Data (FRED) database. The objective was to identify the forecasting model with the best predictive accuracy and generate a 24-month forecast to support financial planning and liquidity management.

## Tools and Technologies

* R
* Time Series Forecasting
* ARIMA Modeling
* ETS Modeling
* Forecast Accuracy Evaluation
* Data Visualization

## Models Evaluated

* Mean Forecast
* Naive Forecast
* Seasonal Naive Forecast
* ETS Models
* ARIMA Models

## Project Files

* `bank_deposit_forecasting_code.R` – R forecasting and analysis code
* `bank_deposit_forecasting_report.pdf` – Final forecasting analysis report
* `bank_deposit_data.csv` – Dataset used for forecasting analysis

## Key Insights

The analysis identified ARIMA and ETS models as the strongest forecasting approaches for predicting U.S. commercial bank deposits. Model evaluation included RMSE, MAE, MAPE, and residual diagnostics to compare forecasting performance.

## Dataset

Monthly commercial bank deposit data sourced from the Federal Reserve Economic Data (FRED) database.
